Creando una nueva rama:
git branch rombreRama

Mostrar la rama en la que estamos:
git branch

Movernos a la nueva rama:
git checkout nombreRama

Copiando o Creando:
git add .

Haciendo Commit:
git commit -m “mensaje”

Push para el main:
git push origin main (main es el nombre de la rama)

Push para el master:
git push origin master (main es el nombre de la rama)

Push para la rama:
git push -u origin nombreRama
